# Security Policy

## Responsible Use

This port scanner is intended for:
- Network administration
- Security auditing with proper authorization
- Educational purposes
- Penetration testing with explicit permission

## Legal Compliance

Users must:
- Only scan networks and systems they own or have written permission to test
- Comply with all applicable local, state, and federal laws
- Respect rate limits and avoid denial of service
- Not use this tool for malicious purposes

## Reporting Security Issues

If you discover a security vulnerability in this tool, please report it by:

1. **DO NOT** open a public issue
2. Email the maintainer directly (add your email here)
3. Include details about the vulnerability
4. Allow time for a fix before public disclosure

## Security Best Practices

When using this scanner:

- Run scans during authorized maintenance windows
- Use appropriate thread counts to avoid overwhelming targets
- Keep logs of scan activities for audit purposes
- Implement rate limiting for production environments
- Monitor for unusual scan patterns that might indicate misuse

Security patches will be released as needed. Keep your copy updated to the latest version.
